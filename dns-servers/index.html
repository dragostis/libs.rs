DNS Servers

Trust Dns

TRust-DNS is a safe and secure DNS server and client with DNSec support.It is working to be a replacement for BIND9. DNSSec on the client side, withNSEC validation for negative records, is complete. The client and server bothsupport dynamic DNS with authenticated requests.
